# Personal Portfolio

This is a portfolio with a cute oreko, made it for myself.

You can browse multiple pages inside the application itself, has a contact form, personal assistant and a lot more!

## Run Locally

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
npm install
```

Add enviroment variables

```bash
Add GEMINI_API_KEY & RESEND_API_KEY
```

Start the server

```bash
npm run start
```
