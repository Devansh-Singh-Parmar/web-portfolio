## Run Locally

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
npm install
```

Add enviroment variables

```bash
Add GEMINI_API_KEY & RESEND_API_KEY
```

Start the server

```bash
npm run start
```
